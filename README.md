## 🚀 What is PrivaNet?

**PrivaNet** is an open-source **Network Intrusion Detection System (NIDS)** that uses **Federated Learning** to detect anomalies without exposing sensitive network traffic to a central server.

- 🧠 **Decentralized AI** – Each site trains locally, only model updates are shared.
- 🔐 **GDPR‑ready** – Raw packet data never leaves your premises.
- ⚡ **Real‑time detection** – Isolation Forest + Lightweight Neural Network.
- 📊 **SIEM Integration** – Native JSON export for Splunk, Elastic, or Grafana.
- 📱 **Alerting** – Slack, Telegram, or Email out-of-the-box.
